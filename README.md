# triangdist

[![CRAN status](https://www.r-pkg.org/badges/version/triangdist)](https://CRAN.R-project.org/package=triangdist)
[![Test Coverage](https://img.shields.io/badge/coverage-100%25-brightgreen)](https://github.com/yourusername/triangdist)
The goal of `triangdist` is to provide a robust, fully vectorized implementation of the **Triangular Distribution** in R. It follows the standard R naming convention for probability distributions: `d`, `p`, `q`, and `r`.

## Key Features

* **100% Test Coverage:** Every line of code is verified through rigorous unit testing.
* **Fully Vectorized:** All functions handle vector inputs for parameters and data points.
* **Safety First:** Includes detailed input validation to prevent mathematical errors.
